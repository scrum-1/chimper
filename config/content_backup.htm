簡介

CMSimfly 分為動態內容管理系統與靜態網頁, 使用者若希望採用動態系統, 可以參考流程說明, 在 Ubuntu Server 中, 利用 uwsgi 與 Nginx 啟動 CMSimfly.

而 CMSimfly 的靜態系統則可以套用

https://github.com/BlackrockDigital

https://themewagon.com/theme_tag/bootstrap-4-templates/

等網站所提供的 Bootstrap 模板, 建立能夠在各種不同螢幕大小流程使用的靜態網頁.
